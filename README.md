# Digiteekaart.ee

Avalik maandumisleht Ettevõtluskeskus OÜ kanalile `digiteekaart.ee`.

Fookus: EIS digitaliseerimise teekaardi toetuse sobivuse eelhindamine ja teekaardi ettevalmistus.

## Arendus

```bash
npm install
npm run dev
npm run build
```
